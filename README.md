## How to Run

1. Install dependencies:
    ```bash
    pip install kivy cryptography
    ```
2. Run the app:
    ```bash
    python3 tulsi_app.py
    ```
3. (Optional) Use the included `start_tulsi.sh` for easy launch on Linux/BSD.
